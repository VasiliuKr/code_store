Клавиши для шторма
Shift + Shift - Поиск по названию всего (обычно для файлов и папок использую, так быстрее))))
Shift + Delete - Удаление строки (как по мне удобнее)
Ctrl + Shift + Backspace - Перенос курсора к прошлой точке редактирования
Ctrl + Alt + J - Оборачивание выделенного текста тегами
Ctrl + Alt + вверх или вниз - Перемещение выделенных строк (или строки) вверх вниз (каверзная штука, иногда хочет думать за тебя)
Ctrl + Shift + Alt + L - prettify code
